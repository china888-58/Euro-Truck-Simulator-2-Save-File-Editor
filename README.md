# 欧卡 / 美卡 存档编辑器

> **Euro Truck Simulator 2** 与 **American Truck Simulator** 存档编辑工具



## 下载

从 [GitHub Release](https://github.com/china888-58/Euro-Truck-Simulator-2-Save-File-Editor/releases/latest) 下载 **`EuroTruckSaveEditor-Setup-v3.0.0.exe`**（约 37 MB 安装包），双击运行按提示完成安装，从开始菜单或桌面快捷方式启动。

> **重要**:从 v1.1.9 起改用 **Inno Setup 安装包**分发,双击安装包即可完成安装。安装包会自动:
> - 注册到「添加/删除程序」(可系统卸载)
> - 创建开始菜单和桌面快捷方式
> - 安装新版本时自动卸载旧版本
> - 程序内检查更新后自动下载安装包并静默安装

> 系统要求:Windows 10 / 11(64 位)。首次运行若被 SmartScreen 拦截,点击「更多信息」→「仍要运行」。

## 支持的存档格式

| 格式 | 说明 | 支持 |
|------|------|------|
| 明文文本 | 已解密的 `.sii` 文本 | ✓ 读写 |
| zlib 二进制 | `0x03` 头压缩 | ✓ 读写 |
| ScsC 加密 | AES-256-CBC + zlib（ETS2 1.5+ 默认） | ✓ 读写 |
| BSII 二进制 | 结构化二进制（ETS2 1.6+ 默认） | ✓ 读写 |
| 加密 BSII | ScsC + BSII（最新版本） | ✓ 读写 |

## 使用方法

1. **备份存档**：建议先复制一份存档文件备用。
2. **安装程序**：双击 `EuroTruckSaveEditor-Setup-v3.0.0.exe` 完成安装。
3. **启动程序**：从开始菜单或桌面快捷方式启动。
4. **打开存档**：点击「📂 打开存档」（或 Ctrl+O）。
   - 默认存档路径：
     - ETS2: `Documents\Euro Truck Simulator 2\profiles\<存档>\save\1\game.sii`
     - ATS: `Documents\American Truck Simulator\profiles\<存档>\save\1\game.sii`
5. **修改数据**：在侧边栏导航的各个页面中调整参数。
6. **保存**：点击「💾 保存」（默认自动备份原文件为 `game.bak.sii`）。
7. **重新进游戏**：保存后建议先**完全退出游戏**再启动，以避免冲突。

### 快捷键
- `Ctrl+O` 打开存档
- `Ctrl+S` 保存
- `Ctrl+Shift+S` 另存为
