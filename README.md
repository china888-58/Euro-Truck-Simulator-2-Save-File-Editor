## 支持的存档格式

| 格式 | 说明 | 支持 |
|------|------|------|
| 明文文本 | 已解密的 `.sii` 文本 | ✓ 读写 |
| zlib 二进制 | `0x03` 头压缩 | ✓ 读写 |
| ScsC 加密 | AES-256-CBC + zlib（ETS2 1.5+ 默认） | ✓ 读写 |
| BSII 二进制 | 结构化二进制（ETS2 1.6+ 默认） | ✓ 读写 |
| 加密 BSII | ScsC + BSII（最新版本） | ✓ 读写 |

## 使用方法

1. **备份存档**：建议先复制一份存档文件备用。
2. **打开存档**：双击 `EuroTruckSaveEditor.exe`，点击「📂 打开存档」。
   - 默认存档路径：`Documents\Euro Truck Simulator 2\profiles\<存档>\save\1\game.sii`
3. **修改数据**：在侧边栏导航的各个页面中调整参数。
4. **保存**：点击「💾 保存」（默认自动备份原文件为 `game.bak.sii`）。
5. **重新进游戏**：保存后建议先**完全退出游戏**再启动，以避免冲突。

## 常见问题

**Q：修改后进游戏没生效？**
A：请确认已**完全退出游戏**（不是最小化），且游戏未在运行时保存。游戏运行中会锁定存档文件。

**Q：车轮磨损显示 2.0000？**
A：那是数组长度（2 个轮子），不是磨损值。本工具会自动读取各轮最大磨损值显示并清零。

**Q：BSII 格式能修改什么？**
A：支持标量字段修改（金钱、等级、燃油、磨损等）。数组字段（如 `accessories[]`、`drivers[]`）在 BSII 格式下不可直接修改，这是二进制结构的限制。

**Q：保存时报错？**
A：请确认文件未被游戏占用，且有写入权限。BSII 格式会保留原始结构，只修改目标字段。

**Q：检查更新提示限流？**
A：GitHub API 未认证每小时限 60 次。程序会自动用本地缓存显示最近一次的 release 信息，不影响使用。
